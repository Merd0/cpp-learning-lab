# STL Foundation

The standard library is a major part of modern C++. Good C++ code often uses
standard containers and algorithms instead of manual loops and raw memory.

## Key Ideas

- `std::vector` is the default dynamic sequence container.
- Algorithms such as `std::sort`, `std::count_if`, and `std::transform` express
  intent clearly.
- Iterators connect containers and algorithms.
- Prefer algorithm names that describe the operation.
